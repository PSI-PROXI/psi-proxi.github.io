More About Proteomics Expression Interface

A simple and efficient way of exchanging protein expression data using distributed repositories. Simple and robust enough to make accessible data from disparate resources using simple tab-separated, XML file format or JSON objects ( Which protein / peptide was identified in a given experimental setting or which proteins was identified in different experiments.)
