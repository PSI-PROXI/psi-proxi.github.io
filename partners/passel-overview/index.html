PASSEL Overview

PASSEL supports the submission of datasets generated by SRM approaches, by storing the experimental results and the corresponding raw data. The submitted raw data is automatically reprocessed in an uniform manner using mQuest, a component of the mProphet software suite, and the results are loaded into the database. The original files and the corresponding reprocessed results are made available to the community. Additionally, the measured transitions are incorporated into the SRMAtlas catalog of transitions. Detailed metadata and structured sample information are in accordance with the PX guidelines.

Data submission and format support

PASSEL uses a web interface for the data submission process and is now supporting the following data types:

  • Study metadata, such as dataset title, submitter contact information, sample source, sample preparation and instrument used.
  • Transition lists describing which transitions were measured for each peptide and targeted ions, along with optional supporting information (collision energy, expected retention time and expected relative intensities). This information is available in a tab-separated file or in the standard TraML format.
  • Mass spectrometer output files in mzML or mzXML format. If these are not available, the vendor formats .wiff (AB SCIEX), .raw (Thermo) or .d (Agilent) are also supported. All the files supplied by the users are uploaded to a specially created FTP account and they can be browsed using the ‘PASSEL Experiment Browser’.

Data Mining and visualization

The PASSEL Experiment Browser enables filtering based on fields like research contact information, organism, sample and instrument type. As mentioned above, it is also possible to download the original files provided. The PASSEL Data Browser provides a description of each selected transition group and the data collected for it, a link to visualize the trace group using the Chromavis chromatogram viewer and further links to the information available in SRMAtlas and PeptideAtlas, in order to extract or compare spectral features of the targeted peptides.
